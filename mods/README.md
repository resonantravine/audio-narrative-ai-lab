# Cola MODs

This directory contains Cola MODs (switchable agent personas) developed in this lab.

## Available MODs

### Good Night Mode

A low-stimulation bedtime persona. Instead of engaging, it gradually withdraws — helping the user let go of the day and move toward sleep.

- [AGENT.md](good-night/AGENT.md)
- Based on the [Good Night](https://github.com/resonantravine/good-night-voice-experience) voice interaction study

## How to install

```bash
# Clone or copy the MOD
cp -r mods/good-night ~/.cola/mods/

# Switch to it in Cola
# Say: "切换到 good night mode"
# Or use the mod_switch tool
```
